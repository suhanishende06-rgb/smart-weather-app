# Smart Weather App

Smart Weather App is a responsive and user-friendly web application that provides real-time weather information using city search and current location (GPS). The project is built with HTML, CSS, and Vanilla JavaScript using the OpenWeatherMap API.

---

## Features

- Search weather by city name
- Get weather using current location (GPS)
- Real-time temperature in Celsius
- Weather condition with dynamic icons
- Humidity and wind speed display
- Smooth UI animations
- Clean and responsive design

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- OpenWeatherMap API

---

## How to Use

1. Enter a city name and click on **Search**
2. Or click on **Use Current Location** to fetch weather automatically
3. View temperature, condition, humidity, and wind speed instantly

---

## API Setup

This project uses the OpenWeatherMap API.

Steps to get API key:
1. Visit openweathermap.org
2. Create a free account
3. Generate an API key
4. Paste the key in `script.js`

- Shows message when city is not found
- Handles browser geolocation permission errors
